Imagining Physics simple static site

Open this folder in VS Code and run index.html with the Live Server extension.

Pages included:
- index.html: simple landing page with YouTube channel link
- videos.html: long-form YouTube videos, with newest available/listed cards at the top
- shorts.html: YouTube Shorts
- interactive.html: GitHub Pages / browser-based demos
- shadertoy.html: Shadertoy shader links from the Amirk profile/search results
- games.html: Game of Coriolis and VR version links

Update notes for this version:
- Added Bohmian3DBox, BohmianDoubleSlit, BohmianSternGerlach3D-public, and BohmianTunneling deployment link to Interactive demos.
- Added the Amirk Shadertoy profile link and every individual Amirk shader link that search exposed reliably.
- Compact preview blocks are used where screenshots are not available yet. Replace them with images later if you want.

For YouTube cards, replace VIDEO_ID-like links or thumbnails by copying an existing card and changing the title, description, and YouTube ID.
